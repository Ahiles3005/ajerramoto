# AJERRA Moto — Интернет-магазин эндуро мотоциклов и запчастей

Современный интернет-магазин для продажи эндуро мотоциклов, запчастей и аксессуаров. Проект разработан с использованием PHP, CSS и JavaScript, с интеграцией Swiper для слайдеров.

## 🛠 Технологии

- **Backend:** PHP 8.x
- **Frontend:** HTML5, CSS3, JavaScript (ES6+)
- **Слайдеры:** Swiper 12
- **Шрифты:** Inter, Krona One (Google Fonts)
- **Сборка:** Без сборщика, нативный PHP

## 📁 Структура проекта

```
ajerra/
├── includes/              # Повторно используемые компоненты
│   ├── header-home.php    # Шапка для главной страницы (светлая)
│   ├── header-page.php    # Шапка для внутренних страниц (тёмная)
│   └── footer.php         # Подвал сайта
├── public/                # Статические файлы
│   └── assets/            # Изображения, иконки, SVG
├── src/                   # Исходные файлы стилей
│   └── styles/            # CSS файлы для каждой страницы
├── cart.php               # Страница корзины
├── category.php           # Страница каталога/категории
├── index.php              # Главная страница
├── order.php              # Страница оформления заказа
├── order-success.php      # Страница успешного заказа
└── product.php            # Страница товара
```

## 📄 Описание страниц

### Главная страница (`index.php`)
- Hero-слайдер с преимуществами
- Секция моделей мотоциклов (сетка товаров)
- Секция запчастей с двумя Swiper-каруселями (Популярное / Новинки)
- Секция "О нас"
- Контакты и карта магазинов

### Каталог (`category.php`)
- Боковая панель с фильтрами и категориями
- Сетка товаров с пагинацией
- Swiper-слайдер похожих товаров

### Страница товара (`product.php`)
- Swiper-галерея изображений товара
- Информация о товаре (цена, статус, описание)
- Характеристики
- Секция "Похожие товары" со Swiper
- Вкладки (Описание, Характеристики, Отзывы, Оплата, Доставка)

### Корзина (`cart.php`)
- Список товаров с управлением количеством
- Кнопки удаления товаров
- Кнопки "Купить в один клик" и "Перейти к оформлению"
- Состояние пустой корзины с иллюстрацией

### Оформление заказа (`order.php`)
- Форма данных покупателя
- Выбор способа доставки и оплаты
- Переключатели (toggles) для согласия с офертой

### Успешный заказ (`order-success.php`)
- Иллюстрация успешного заказа
- Информация о номере заказа

## 🎨 Система стилей

### Основные CSS-файлы
- `main.css` — глобальные стили, переменные, компоненты
- `cart.css` — стили страницы корзины
- `order.css` — стили страницы оформления заказа
- `product.css` — стили страницы товара
- `category.css` — стили страницы каталога

### CSS-переменные
```css
:root {
  --color-primary: #FEB05D;        /* Основной акцентный цвет */
  --color-primary-dark: #E09645;   /* Тёмный вариант основного */
  --color-text: #242424;           /* Основной цвет текста */
  --color-text-muted: #777;        /* Приглушённый текст */
  --color-card: #fff;              /* Фон карточек */
  --color-bg-dark: #1a1a1a;        /* Тёмный фон */
  --radius-sm: 8px;                /* Малый радиус скругления */
  --radius-md: 16px;               /* Средний радиус скругления */
}
```

### Адаптивность
- Desktop: > 1200px
- Tablet: 768px — 1200px
- Mobile: < 768px

## 🚀 Установка

1. Клонируйте репозиторий:
```bash
git clone https://github.com/username/ajerra.git
cd ajerra
```

2. Настройте веб-сервер (Apache/Nginx) на папку проекта

3. Убедитесь, что установлен PHP 8.x

4. Откройте сайт в браузере

## 📝 Разработка

### Добавление новой страницы

1. Создайте PHP-файл в корне проекта
2. Подключите шапку и подвал:
```php
<?php include 'includes/header-page.php'; ?>
<!-- Контент страницы -->
<?php include 'includes/footer.php'; ?>
```

3. Создайте CSS-файл в `src/styles/`
4. Подключите стили в `<head>`

### Добавление нового компонента

1. Используйте существующие CSS-переменные
2. Следуйте принятой структуре классов (BEM-подобная)
3. Добавьте адаптивные стили в media queries
